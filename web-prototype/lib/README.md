# lib フォルダ（Live2D Cubism Core）

ここに **`live2dcubismcore.min.js`** を置いてください（あなたがモデルと一緒に持っているファイル）。

- `index.html` は `lib/live2dcubismcore.min.js` を読み込みます。
- PIXI と pixi-live2d-display は、ブラウザ確認用に CDN から読み込んでいます
  （オフラインのiOSアプリに組み込む段階で、これらもローカルに同梱します）。

## 追加方法
`web-prototype/model/README.md` と同じ手順（GitHub Desktop か git）で、
`live2dcubismcore.min.js` をこの `lib/` フォルダに入れて Push してください。
